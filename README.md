### identidade, Acesso, e Segurança

* Microsoft entra Id: serviço de gerenciamento de identidades e acesso baseado em nuvem do microsoft

	- Autenticação
	- Log on (SSO)

* Autorização: determina nível de acesso de uma pessoa/ recurso autenticado
